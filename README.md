# PHP

#### 8.4-fpm

```shell
cd 8.4-fpm
docker build -t brighterly/php:8.4-fpm .
docker run -it brighterly/php:8.4-fpm bash
```

#### 8.4-fpm-alpine

```shell
cd 8.4-fpm-alpine
docker build -t brighterly/php:8.4-fpm-alpine .
docker run -it brighterly/php:8.4-fpm-alpine bash
```
